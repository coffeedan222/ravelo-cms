## getEnv() ⇒ <code>String</code>
Grabs the environment from the
NODE_ENV environment variable.
Valid environments are:
 - production
 - development
 - test

Throws an error if environment
value is invalid.

**Kind**: global function  
**Returns**: <code>String</code> - name of environment
